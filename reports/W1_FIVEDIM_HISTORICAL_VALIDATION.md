# W1_FIVEDIM_HISTORICAL_VALIDATION — 阶段 C 因子有效性审计

**日期**: 2026-06-17
**性质**: 纯研究 · 只读本地数据 · 不接 API · 不动模型/λ · 不写任何因子系数
**问题**: FiveDim 的候选因子(ELO/近况/净胜/射正率/xG/休息天数)是否在**市场之外**带来增量预测信息?
**一句话答案**: **没有。** 在所有有统计功效的分层里,这些因子相对市场基线的**留出集**改善 ≈ 0。

---

## 1. 方法

- **样本**(只读本地、leakage-safe,详见 `W1_FACTOR_LAMBDA_DATA_INVENTORY_V1`):每场一行 = 市场隐含 1X2 概率 + **仅用该场之前**的滚动因子 + 真实赛果。`scripts/w1_factor_sample_builder.py` 顺序构造、emit 在 update 之前;`scripts/check_w1_factor_sample.py` 用"首次出场必无历史"做**结构性防泄漏证明**,PASS。
- **检验**:每层各自,多项 logistic(numpy 实现)。M0 = 截距 + 市场对数概率(复现市场);M1 = M0 + 单个因子(训练集标准化)。**训练集拟合、留出集评估**;Δlog-loss < 0 = 因子带来市场未定价的增量信息。
- **分层严格分开**,不外推(league / wc_qualifier / wc_finals)。
- 样本量:**11,814 行**(联赛 10,733 / 预选赛 889 / 决赛圈 192)。

---

## 2. 结果(留出集 Δlog-loss vs 市场基线;负=改善)

| 因子 | 联赛(留出 n=3473, 市场LL=0.971) | 预选赛(n=178, 0.821) | 决赛圈(n=31, 0.943) |
|---|---|---|---|
| ELO 差 | **−0.0003** | +0.0052 | −0.0009 |
| 近况积分差 | +0.0001 | +0.0030 | +0.0011 |
| 滚动净胜差 | +0.0008 | −0.0002 | −0.0089 |
| 射正率差 | +0.0001 | +0.0024 | +0.0043 |
| 滚动 xG 差 | 数据不足(联赛无xG) | +0.0125 | 数据不足 |
| 休息天数差 | +0.0027 | +0.0007 | +0.0048 |

**读法**:有功效的两层(联赛 3,473 / 预选赛 178)里,**没有任何因子的改善超过 0.0003 nat**——联赛 ELO 的 −0.0003 是基线的 0.03%,落在噪声里。决赛圈个别为负(净胜 −0.0089)但**留出仅 31 场**,无统计功效,等同噪声。

---

## 3. 结论与解释

1. **候选因子不构成相对市场的独立优势。** W1 的市场隐含基线已经把 ELO/近况/净胜/射门隐含进价格;再叠这些因子,留出集上**不改善校准**。这与 W1 的本质一致:它是**市场复述**,不是独立因子模型。
2. **这正是此前反复强调的"高命中 ≠ edge"的实证版**:市场在主流联赛是高效的,公开历史统计很难超越收盘赔率。
3. 决赛圈的"改善"是 31 场的随机起伏,**不可作为上线依据**。

---

## 4. 建议(直接影响后续阶段)

- **不进入阶段 E(因子→λ)。** 数据不支持把这些因子写进 λ;强行写入只会用过拟合换来留出集无改善甚至变差。**保持固定 ρ、市场反解 λ 不变。**
- **阶段 D(confidence 调整)若做,只能"软用":** 因子仅用于**解释/风险旗标/数据质量**(例如"市场与近况背离 → 标注分歧、降可信度"),**不得改概率**。这与已上线的"五维就绪度=非独立优势"口径一致。
- **FiveDim 维持"理解层/解释层"定位**,不升格为"独立预测层"。

---

## 5. 诚实的边界(不夸大这份否定结论)

- 这是**市场效率检验** + 简单模型;"没改善"**不等于**"任何因子永不可能有用"。它只说明:**用当前本地特征集 + 简单叠加,在留出集上拿不到增量**——因此**现在不具备上线 λ 的证据**。
- **未检验的维度**:阵型化学(本地无首发/球员数据)、历史天气(无)——这两维**无法验证**,不在结论范围。
- 决赛圈 n=192/留出 31,**功效不足**,只给方向性。
- xG 仅国际 312 场可用、联赛无,xG 结论弱。
